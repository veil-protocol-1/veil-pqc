# x402-pqc Community Specification Charter

## Project Name
x402-pqc — Quantum-Resistant HTTP Payment Standard

## Scope
Define a quantum-resistant extension to the x402 HTTP payment protocol
using NIST-standardized post-quantum cryptography (ML-DSA-65, ML-KEM-768),
suitable for AI agents, autonomous systems, and human wallets on
EVM-compatible networks.

## Governance
Maintained by Veil Foundation under Linux Foundation Community Specification
License. Contributions welcome via pull request. Specification decisions
made by Founding Architect with community input.

## Intellectual Property
Apache 2.0. All contributors must sign the Linux Foundation CLA.

## Deliverables
1. x402-pqc specification document
2. Reference implementation in TypeScript (@veil/x402-pqc)
3. Conformance test suite
4. Migration guide from x402 ECDSA

## Versioning
Semantic versioning. Current: 1.0.0-draft
